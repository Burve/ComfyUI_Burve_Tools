# March 2026 Crop + Mask Workflow Release

## Release Summary

- Added `Burve Crop + Mask Load`, an interactive crop-and-mask loader with a custom editor.
- Added `Burve Crop + Mask Apply`, which places a replacement image back into the full-size source using the crop region and mask.
- Added `aspect_ratio_override` support to both Gemini image-generation nodes so crop-safe aspect ratios can drive the request payload directly.
- Added the `web/js` editor support files that power the crop/mask workflow UI.
